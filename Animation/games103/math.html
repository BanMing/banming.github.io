射线与圆的碰撞

我们要求一个从p点出发的射线是否与以c点为圆心半径为r的圆相交也就是发生碰撞

首先我们假设交点为p(t),同时根据插值公式得到：p(t) = p + vt

我们只需要判断p(t)点到c点的距离是否等于圆心即可，就可以得到

||p(t) - c|| = r

两边在平方

||p(t) - c||² = r²

再根据一个向量的模的平方为该向量点乘自己

(p - c + tv) · (p - c + tv) = r²

根据多项式变形得到

(v · v)t² + 2(p - c) · vt + (p - c) · (p - c) - r² = 0

求解二元一次方程即可，一般解有三种情况：

  • 没有解：无交点
  • 一个解：一个交点
  • 两个解：两个交点

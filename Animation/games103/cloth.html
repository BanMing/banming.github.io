                                        # Newton-Raphson Method

                                        解决非线性系统

                                        限制：函数需要连续

                                        一阶导数就是在描述一个函数的切方向

                                        对一个函数的一阶导数做一个泰勒展开:

                                        F′(x)≈F′(x(k)) +F″(x(k))(x−xk)

                                        也就是一阶导数可以是在k位置的一阶导数加上k位置的二阶导数乘以Δx。

                                        方法流程就是：

                                        从x0开始计算直到Δx足够小时就中断。

                                        函数的一阶导数等于0时，既可以是最大值也可以最小值。可以使用二阶导数来判断是最大值还是最小值：

                                        • F''（x）> 0 ,最小值
                                        • F''（x）< 0 ,最大值
                                        • 如果二阶导数永远大于0，那么函数没有最大值，存在唯一的一个最小值。

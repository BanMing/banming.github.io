ARPG GAS 数据资产参考

这篇梳理一个真实 ARPG 项目里围绕 GAS 的数据资产体系：施法资产、攻击产物资产（近战环 / 飞行物）、GameplayEffect 配置规范，以及 AbilitySet 与标签关系映射。

设计要点：能力本身是空壳，所有"参数 + 数值 + 引用关系"都丢进 DataAsset，让程序员、关卡设计师、平衡策划各自有清晰的工作面。

1. 资产层次结构

加载顺序：SpellAsset → RingAssets + FlightAssets（并行），全部经由内部 AssetManager 异步加载返回 FStreamableHandle。

2. USpellAsset

2.1 基础属性

属性名类型说明
SpellNameFText技能显示名称（本地化）
HitGameplayCuesFGameplayTagContainer命中目标时播放的 GameplayCue 集合
StartGameplayCuesFGameplayTagContainer技能激活时播放的 Cue
EndGameplayCuesFGameplayTagContainer技能结束时播放的 Cue

2.2 飞行物属性

属性名类型说明
SpellFlightsTArray<TSoftObjectPtr<USpellFlightAsset>>该技能关联的所有飞行物（软引用，异步加载）
SpawnFlightOnStartbooltrue 时激活立即生成（否则等 AnimNotify）
FlightAnimNotifyTagOverridesTMap<FGameplayTag, FGameplayTag>AnimNotify → SpellFlight 的标签重定向

2.3 消耗属性

属性名类型说明
CostAmmoint32弹药消耗量
CostStaminaPercentfloat体力消耗百分比（0.0–1.0）

2.4 伤害 FX 类型

属性名类型说明
SpellDamageFXTypeTagFGameplayTag默认伤害特效类型（决定血迹/火花类型）
WeaponIDTagsForDamageFXTypeTagTMap<FGameplayTagContainerKey, FGameplayTag>基于武器标签容器的 FX 类型覆盖

2.5 肢解

属性名类型说明
DismemberBonesOnKillingTArray<FName>致命一击时强制肢解的骨骼（空 = 由 RingAsset 决定）

2.6 近战变体（USpellAssetMelee）

属性名类型说明
ParryWindowTagsFGameplayTagContainer招架窗口期标签

2.7 敌方变体（USpellAssetEnemy）

属性名类型说明
AlertRadiusfloat技能使用时警戒范围半径（cm）
CombatPhaseTagFGameplayTag触发该技能的战斗阶段标签

3. USpellProductAsset（攻击产物基类）

抽象基类，不可直接实例化。集中描述"一次攻击的伤害、碰撞、效果"。

3.1 关键结构体

FSpecialFightResultParam

属性类型说明
HitStopTimefloat命中停顿时长（秒）
StaggerTagFGameplayTag触发受击硬直的标签
DismemberBoneFName触发肢解的骨骼名称
BreakThresholdMultiplierfloatBreak 属性阈值乘数
KnockOutTagFGameplayTag触发击倒效果的标签
HitDirectionEHitDirectionType方向计算方式：AttackerToVictim / RuntimeLocationTrace / VectorInAttackerSpace
HitDirectionVectorFVector与 VectorInAttackerSpace 模式配合的本地空间方向向量
BulletPhysicsParamFBulletPhysicsParam弹孔/物理子弹参数

FFightTargetFilter

属性类型说明
TargetTagQueryFGameplayTagQuery目标必须满足的标签查询
MaxRadiusfloat最大作用半径（cm，0 = 无限制）
MaxAnglefloat相对攻击者朝向的最大角度（度）
bCanHitSelfbool是否可命中自身
bCanHitFriendlybool是否可命中友方
bCanHitNeutralbool是否可命中中立目标
ApplicationChancefloat效果应用概率（0.0–1.0）

FSpellEffectSetting

属性类型说明
EffectClassTSubclassOf<UGameplayEffect>要应用的 GE 类
SetByCallerOverridesTMap<FGameplayTag, float>覆盖 GE 的 SetByCaller 幅度
DamageFXTypeOverrideFGameplayTag覆盖该效果的伤害特效类型

FCollisionTraceInterfaceParam

属性类型说明
TraceShapeETraceShapeSphere / Box / Capsule
TraceRadiusfloat球形/胶囊形半径（cm）
TraceHalfExtentFVector盒形检测半扩展
TraceChannelECollisionChannel检测使用的碰撞通道
MaxHitCountint32单次检测最大命中目标数

3.2 USpellProductAsset 主要字段

属性类型说明

BuffEffectsTArray<FSpellEffectSetting>Buff 效果

4. USpellRingAsset（近战环）

继承自 USpellProductAsset。

4.1 碰撞检测

属性类型说明

4.2 检测行为

属性类型说明

4.4 HitStop

属性类型说明

HitStopAffectsTargetbool是否影响目标

6. GameplayEffect 配置规范

6.1 GE 类型选择

类型持续适用推荐目录
Instant瞬时伤害、一次性资源消耗Content/<root>/GE/Damage/
Duration固定时长护盾、临时增益Content/<root>/GE/Buff/
Infinite永久（手动移除）被动 Buff、装备效果Content/<root>/GE/Passive/

6.2 Instant 伤害 GE

GE 名称：GE_MeleeDamage
Duration Policy: Instant

[Modifiers]
Attribute: AttributeSetBase.MetaDelta
    Modifier Op: Add
    Magnitude: SetByCaller
    SetByCaller Tag: Damage.Physical
    SetByCaller 默认值: -100.0  (负值 = 减少属性)

[Tags]
Asset Tags: GameplayEffect.Damage.Physical

重要： 伤害 GE 的 Modifier 目标属性必须是 MetaDelta，而不是 Health 等最终属性。MetaDelta 是所有属性集的中间缓冲，在 PostGameplayEffectExecute 中被读取并应用到实际属性上，从而实现 Clamp 和额外逻辑。

6.3 Duration Buff GE

GE 名称：GE_DefenseUp_10s
Duration Policy: Has Duration
Duration Magnitude: 10.0 s
Period: 0

[Modifiers]
Attribute: AS_Defense.DefenseRating
    Modifier Op: Multiply
    Magnitude: Scalable Float = 1.5  (+50% 防御)

[Tags]
Asset Tags: GameplayEffect.Buff.Defense
Applied Tags: OwnedTagsCategory.BuffState.DefenseUp

6.4 Infinite 被动 GE

GE 名称：GE_Passive_StaminaRegen
Duration Policy: Infinite
Period: 1.0 s
bExecutePeriodicEffectOnApplication: true

[Modifiers]
Attribute: AS_Stamina.MetaDelta
    Modifier Op: Add
    Magnitude: SetByCaller
    SetByCaller Tag: Stamina.RegenAmount
    默认值: 5.0

6.5 Cost GE

GE 名称：GE_Cost_Stamina
Duration Policy: Instant

[Modifiers]
Attribute: AS_Stamina.MetaDelta
    Modifier Op: Add
    Magnitude: SetByCaller
    SetByCaller Tag: Cost.Stamina
    默认值: -20.0

由 UGameplayAbilityProject::CheckCost() 在激活前验证：Stamina 不足时技能无法激活，激活时通过 ApplyGameplayEffectToSelf 实际扣除。

6.6 SetByCaller 常用标签

标签含义典型值范围
Damage.Physical物理伤害量50–500
Damage.BreakDamage破防伤害10–100
Damage.PoiseBreak架势破坏5–50
Damage.StaggerDamage硬直积累10–80
Cost.Stamina体力消耗（负值）10–50
Cost.Ammo弹药消耗（由 CustomCost 处理）1–10
Heal.Health回复生命20–200
Stamina.RegenAmount每次体力回复3–10

9. 数据驱动设计规范

9.1 新技能完整资产创建顺序

1. 创建 SpellAsset
   路径：Content/<root>/Spell/<WeaponType>/
   命名：DA_<WeaponType>_<AttackName>_Spell
   → 配置蒙太奇引用、Cue、飞行物引用

2. 创建 SpellRingAsset（近战）
   路径：Content/<root>/SpellRing/<WeaponType>/
   命名：DA_<WeaponType>_<AttackName>_Ring
   → 配置 Socket、碰撞半径、目标 GE 类

   或 创建 SpellFlightAsset（远程）
   路径：Content/<root>/SpellFlight/<WeaponType>/
   命名：DA_<WeaponType>_<AttackName>_Flight
   → 配置速度、存活时长、命中 GE 类

3. 创建 GameplayEffect
   路径：Content/<root>/GE/
   命名：GE_<WeaponType>_<AttackName>_Damage
   → MetaDelta Modifier + SetByCaller 标签

4. 在 SpellRing/SpellFlight 中引用 GE
5. 在 Ability Blueprint 中引用 SpellAsset
6. 在 AbilitySet 中注册 Ability

9.2 命名规范

资产类型前缀示例
SpellAssetDA_DA_Katana_LightAttack_Spell
SpellRingAssetDA_DA_Katana_LightAttack_Ring
SpellFlightAssetDA_DA_Revolver_BasicShot_Flight
GameplayEffectGE_GE_KatanaDamage_Physical
AbilitySetDA_DA_AbilitySet_Katana
TagRelationshipDA_DA_TagRelationship_PlayerCombat

9.3 SetByCaller 数值平衡工作流

  1. 在 SpellProductAsset 的 SetByCallerMagnitudes 中定义基础伤害值
  2. GE 中声明对应 SetByCaller 标签但不设默认值（强制由调用者传入）
  3. 技能执行流通过 MakeOutgoingGameplayEffectSpec + AssignTagSetByCallerMagnitude 传递
  4. 数值调整只需修改 SpellProductAsset，无需重编译

9.4 软引用规范

  • SpellAsset → SpellFlightAsset：必须 TSoftObjectPtr<> + 异步加载
  • SpellRingAsset → GE 类：可硬引用（Ring 激活前 SpellAsset 已加载）
  • Mesh / Niagara / Sound 资产：全部软引用，由 AssetManager 按需加载

这套数据资产体系最大的好处在于：程序员只维护类与字段，关卡/策划只维护 DataAsset 实例——能力数量增长不再需要程序员持续介入。

8. GE 应用完整流程

创建 Spec

// 方式一：从 Ability（推荐，自动继承 Ability 标签）
FGameplayEffectSpecHandle Spec =
    Ability->MakeOutgoingGameplayEffectSpec(GEClass, Level);

// 方式二：从 ASC
FGameplayEffectSpecHandle Spec =
    ASC->MakeOutgoingSpec(GEClass, Level, ASC->MakeEffectContext());

填充附加信息

// 战斗记录
auto* Ctx = FGameplayEffectContextProject::ExtractEffectContext(Spec.Data->GetContext());
Ctx->FightRecord = FightRecord;
Ctx->RelatedAsset = SpellRingAsset;

// SetByCaller（伤害数值由外部注入）
Spec.Data->SetSetByCallerMagnitude(DamageTag, DamageValue);

// 攻击定义（武器、法术参数）
UCombatPayloadFunctionLibrary::AddAttackDefinitionToGameplayEffectSpec(Spec, AttackAsset);

// 命中结果
UAbilitySystemBlueprintLibrary::EffectContextAddHitResult(
    Spec.Data->GetEffectContext(), HitResult, true);

应用

// 应用到目标
SourceASC->BP_ApplyGameplayEffectSpecToTarget(Spec, TargetASC);

// 或应用到自身（成本 GE）
SourceASC->ApplyGameplayEffectToSelf(GEClass, Level, Context);

Duration 类型一览

类型行为适用
Instant修改 BaseValue → 立即结束一次性伤害、回复
Duration持续修改 → 时间到自动移除减速、临时 Buff
Infinite必须手动移除装备 Buff、被动状态
Period周期触发 Instant 效果持续中毒、持续回血

调试时，按这条主线找断点：

  1. 输入没生效 → ProcessAbilityInput
  2. 激活失败 → CanActivateAbility、ApplyAbilityBlockAndCancelTags
  3. 伤害不进 → RingOnHit → MakeOutgoingGameplayEffectSpec → BP_ApplyGameplayEffectSpecToTarget
  4. 属性没变 → PostGameplayEffectExecute（MetaDelta 是否归零、状态是否触发）
  5. 没特效 → ExecuteGameplayCueOnOwner（CueTag 是否注册、GCN 是否被发现）

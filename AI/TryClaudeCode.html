Claude Code Game Studios 技能命令系统

Claude Code Game Studios

这个系统中的技能命令都是通过斜杠(/)开头的命令来调用的。以下是主要的技能命令及其使用方法：

入门命令

  • /start - 启动系统，它会询问您的项目状态，引导您进入适合的工作流程
  • /help - 分析当前状态并提供下一步建议
  • /brainstorm - 从头开始探索游戏创意
  • /project-stage-detect - 分析现有项目的阶段

项目设置与规划

  • /setup-engine [引擎名称] [版本] - 例如：/setup-engine godot 4.6
  • /create-epics - 创建项目史诗级任务
  • /create-stories - 创建具体开发任务
  • /sprint-plan - 规划开发冲刺
  • /estimate - 估计任务工作量

设计相关

  • /quick-design - 快速设计概念
  • /art-bible - 创建艺术风格指南
  • /design-system - 制定设计系统
  • /architecture-decision - 做出架构决策
  • /map-systems - 映射游戏系统

开发相关

  • /prototype - 创建原型
  • /dev-story - 开发具体任务
  • /code-review - 代码审查
  • /hotfix - 修复紧急问题
  • /tech-debt - 处理技术债务

团队协作

  • 团队特定命令：/team-audio, /team-combat, /team-level, /team-narrative, /team-ui, /team-qa等
  • /onboard - 加入新成员到项目

质量保证

  • /qa-plan - 制定质量保证计划
  • /bug-report - 报告bug
  • /bug-triage - 对bug进行分类处理
  • /smoke-check - 进行冒烟测试
  • /playtest-report - 生成游玩测试报告

版本发布

  • /release-checklist - 版本发布检查清单
  • /patch-notes - 生成补丁说明
  • /day-one-patch - 制作首日补丁

审查与评估

  • /design-review - 设计审查
  • /architecture-review - 架构审查
  • /milestone-review - 里程碑审查
  • /retrospective - 项目回顾

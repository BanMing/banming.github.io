强制关闭exe的命令

Start taskkill /im exe_name.exe /f

因为在此系统上禁止运行脚本”解决办法

我们通过管理员权限运行power shell，然后输入命令

 set-ExecutionPolicy RemoteSigned

如果执行了操作后，还出现了 "未对文件 xxxxx 进行数字签名", 需要选择xxx.ps1文件右键 -> 属性 -> 点击 "解除锁定" 框框
